# H100 2B Readiness

This directory is reserved for lightweight H100 readiness evidence generated by:

```text
notebooks/colab_h100_2b_readiness.ipynb
```

No H100 experiment has been run or claimed in this repository snapshot. A valid
report bundle must contain measured:

- `300m_probe.json`
- `1b_probe.json`
- `2b_probe.json`
- `readiness_summary.json`

Each larger probe is admitted only after the previous report's acceptance gate
passes. Do not add checkpoints, optimizer state, packed token shards, tokenizer
training corpora, activation caches, or model weights.

This folder is a readiness target, not a measured H100 feasibility claim.
